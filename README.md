# 💎 Lírio Branco - Sistema ERP e PDV (Versão Demo)

Sistema completo de gestão comercial desenvolvido para uma loja de acessórios. Esta é uma versão de **Demonstração para Portfólio**, onde os dados são salvos localmente no navegador (LocalStorage) para preservar a integridade do banco de dados real em produção.

## 🚀 Tecnologias Utilizadas
* **JavaScript (ES6+)** - Manipulação de dados e regras de negócio.
* **HTML5 & CSS3** - Interface responsiva e design system customizado.
* **Bibliotecas Externas:** SweetAlert2 (Modais), Toastify (Notificações), jsPDF e SheetJS (Exportação de relatórios).

## 🌟 Funcionalidades
* **Gestão de Estoque:** Cadastro de produtos com variação de materiais, fotos e controle de quantidades.
* **Carrinho de Vendas (PDV):** Interface rápida para adicionar produtos, calcular subtotais e fechar pedidos.
* **Dashboard Financeiro:** Cálculo automático de lucros, receitas e despesas.
* **Geração de Recibos:** Exportação de notinhas de compra em imagem e PDF diretamente pelo navegador.
